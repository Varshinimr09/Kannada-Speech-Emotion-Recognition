  In this paper, we have experimenting to identifying the Kannada emotional 
speech signals recognition  using deep learning technique. Speech Emotion 
Recognition (SER) is a hot research topic in the field of HCI (Human Computer 
Interaction). To experiment we have created our own Kannada Emotional 
Speech corpus recording in different android mobile phone, wi th sampling rate 
48 kbps and mono channel and speakers are from the Karnataka and Tamil Nadu 
boarder region, whose accents is a mixer of Kannada, Tamil and Telugu 
languages. The emotions are anger, fear, sad and neutral is considered. Totally we 
have 440 K annada emotional speech signals. To use this speech signal data in our 
experiment, we have first down sampled all speech signals into 16 kbps, mono 
channel and stored as wav files. Then we pass the each sample signals into the 
pre-emphasize phase, then fra ming and windowing is done, then we extracted 
features from each voice frame signal like MFCC, Chroma and „Mel‟ and then 
passed to CNN (Convolution Neural Network) to train and testing the voice 
signals. The CNN is capable to handle small amount of data fo r training, and with 
smaller number of parameters. The average Kannada emotion recognition 
accuracy rate of 93.77% is achieved. All computations are done using Python 
programming language and used Python libraries like Librosa, Keras, pyaudio, 
soundfile, s klearn.

 LITERATURE  SURVEY  
Bennilo Fernandes (2021) et al., have evaluated using 
RNN with hierarchal of LSTM and BiLSTM with 
dropout layers and compared their performances on 
interchanging the layers for a reduced Tamil 
emotional speech data set. Also they evaluated 
different archit ectures like; DHLL, DHLB, DHBL and 
DHBB with dropout layers and the evaluation were 
accuracy, loss, training time and evaluation time. The 
results showed that the DHLB performs better than 
other models. Accuracy rate of 84% is achieved with 
minimum loss in  each seven basic emotions and time 
taken for training and evaluation is also less than the 
other models. Thus, they recommended for the 
reduced Tamil emotional speech data set is to use 
DHLB, since it has returned good efficiency of 
recognition values wit hin an acceptable running time. 
Javier G. Razuri (2015) et al., experimented on the 
human‟s nonverbal cues such as pitch, loudness, 
spectrum, speech rate which are the efficient carriers 
of emotions.  Linear predictive coding (LPC) is used to

such as the vocal tract transfer function and the 
formant frequencies. It has good intuitive 
interpretation both in time domain and in frequency 
domain. The cepstral representation (Linear 
Prediction Cepstral Coeffici ents (LPCC)) of its 
coefficients is more used because of its higher 
retrieval efficiency. Chunyi Wang, presented acoustic 
features typically utilized in speech emotion

extracted from every frame. Time -domain feature is 
the direct process of the time domain waveform. This 
is the most prevalent feature extraction method since 
it is relatively easy to think of. It can extract features 
such as short -time zero crossing rate, short -time 
energy and pitch frequency. Frequency domain

METHODOLOGY  
 
Figure 1: Show the Proposed Model for down sampling our Kannada emotional speech signal, pre emphasizing

In this experiment, each Kannada emotional speech 
signal sample is stored as wav file, and which is down 
sampled at 16000 Hz. Then signal is passed wit h pre -
emphasis phase, here we observe the spectrum of 
voiced speech segment that is high frequencies 
usually have smaller amplitude of high frequency 
bands and decrease the amplitude of the lower bands 
making the spectrum flatter. Then, frame blocking 
whic h refers to the process of segmenting the speech 
signal after pre -emphasis into number of small 
overlapping frames. We have taken frame size of 20 
msec, with an overlap of 6.5 msec [19]. Then each 
frame is passed through the Hamming Window, 
which extracts the stationary part of speech 
[15][17][19]. Next is features extraction stage, here we 
have extracting MFCC ( Mel Frequency Cepstral 
Coefficient) represents the short -term power 
spectrum of a sound, Chroma which  concerns to the 
12 different pitch classes a nd „Mel‟ which is  Mel 
Spectrogram Frequency. We extracted 26 features 
values from each voice signal frame [15][16][17][19], which is passed to the CNN model for training the 
data. Her parameter setting is required before passing 
to CNN, those modification s are  
Data tensor matrix in our experiment : After pre -
processing, we obtain N frames of each voice sample 
and 13 MFCCs, 12 pitchs and one „Mel” value for each 
frame. Here each following frame is dependent on 
previous frame, which is termed as time depende ncy 
or temporal nature. For each voice samples, we have 
2-D data matrix where rows represent contents of 
each frame (13 MFCC + 12 Pitchs + 1 Mel = 26 values) 
and columns represent number of frames that are 
dependent sequentially. So matrix size will be 26 * N 
for each instance of voice sample.  
Hyper Parameters : Then we need tuning of hyper 
parameters to achieve desired accuracy. Here we have 
directly used Keras library which gives flexibility of 
tuning of hyper parameters. Firstly, batch size used is 
32 wh ich is default batch size of Keras then applied 
Adam optimizer. Then Dropout regularization, which 
deals with over fitting in neural networks, here we

232 DISCUSSION AND CONCLUSION  
In our experiment we have used only 440 emotional 
Kannada speech signal of only one regional accents 
and trained and tested, hence we have received only 
93.77% of accuracy when the epochs is 400. If 
consider the all Kannada accents and trained with the 
more number of samples s ignals than may expect the 
more accuracy with less epochs and system can be 
used emotional detection in real world applications. 
In this experiment we have used the feature like 
MFCC, Pitch and Mel. The MFCC are claimed to be 
robust of all the features for  any speech tasks. The 
consonant -vowel transition region also yields emotion 
recognition but when formants features combined 
with the basic spectral features have always 
improvement in the system performance. We 
absorbed that CNN technique is better techni que in 
training and testing for recognition of any Kannada 
emotional speech signals. But only thing is that we 
need to apply all types of Kannada accents for training 
to build Kannada emotion recognition for any 
Kannada accents.
